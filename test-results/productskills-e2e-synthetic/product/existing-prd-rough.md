# Rough PRD: Evidence-Linked PRD Review

## Objective

Help PMs make better PRDs faster.

## Customer

PMs at B2B SaaS companies.

## Problem

PMs have too many notes and need to write PRDs. Existing tools are scattered.

## Proposed Solution

Add an "Evidence Review" mode that creates a PRD from imported notes and lets the user send it to Notion.

## Scope

- Import evidence.
- Create PRD.
- Send to Notion.

## Metrics

- More PRDs created.
- Faster planning.

## Risks

- Integrations might be hard.

## Open Questions

- Which customers need this most?
- What evidence is enough?
