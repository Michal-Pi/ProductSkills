# Retention Analysis

Use when users activate but do not continue using the product.

## Steps

1. Define retained behavior and time window.
2. Segment by use case, acquisition source, plan, persona, or activation path.
3. Identify value frequency and habit or workflow fit.
4. Separate product value issues from reminder, lifecycle, or segmentation issues.
5. Propose retention experiments.
6. Define success metric and guardrail.

## Output

Return retention definition, segment, root-cause hypotheses, experiments, and decision thresholds.
