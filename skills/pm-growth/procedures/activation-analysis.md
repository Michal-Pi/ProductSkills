# Activation Analysis

Use when users sign up but do not reach meaningful first value.

## Steps

1. Define activation event and segment.
2. Map steps from signup to first value.
3. Identify drop-offs, confusion, delays, missing intent, or value mismatch.
4. Separate onboarding UX issues from acquisition quality problems.
5. Propose activation experiments.
6. Define success metric, guardrail, and decision rule.

## Output

Return activation definition, bottleneck, evidence, hypotheses, experiments, and metrics.
